## Descrição
O score de uma string é definido como a soma das diferenças absolutas entre os valores ASCII de cada par de caracteres adjacentes.

Para cada posição `i`, calculamos:
|ord(s[i]) - ord(s[i+1])|
e acumulamos esse valor no total.  
Percorrendo a string uma única vez, conseguimos obter o resultado de forma eficiente.

---

## Solução (Python)
```python
class Solution:
    def scoreOfString(self, s: str) -> int:
        x = 0
        for i in range(len(s) - 1):
            z = abs(ord(s[i]) - ord(s[i + 1]))
            x += z
        return x
