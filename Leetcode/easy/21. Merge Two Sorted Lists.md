# 21. Merge Two Sorted Lists

**Data:** 20/01  
**Dificuldade:** Easy  
**Tópicos:** Linked List, Two Pointers, Recursion, Iteration  
**Linguagem:** Python  
**Link:** https://leetcode.com/problems/merge-two-sorted-lists/description/  

## Descrição
O problema consiste em unir duas listas ligadas ordenadas em apenas uma lista também ordenada.  
Devemos percorrer ambas simultaneamente e, a cada passo, escolher o menor valor entre os nós atuais para construir a nova lista.

O uso de um nó fictício (*dummy node*) facilita o processo, pois evita a necessidade de tratar o primeiro nó como um caso especial.

## Solução (Python)
```python
class Solution:
    def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
        # Dummy node to simplify handling the head of the merged list
        dummy = ListNode(0)
        current = dummy

        # Merge both lists while they have nodes
        while list1 and list2:
            if list1.val <= list2.val:
                current.next = list1
                list1 = list1.next
            else:
                current.next = list2
                list2 = list2.next
            current = current.next

        # Attach the remaining nodes from either list
        current.next = list1 or list2

        # Return the head of the merged list
        return dummy.next
