# 3512. Minimum Operations to Make Array Sum Divisible by K

**Link:** [Minimum Operations to Make Array Sum Divisible by K - LeetCode](https://leetcode.com/problems/minimum-operations-to-make-array-sum-divisible-by-k/description/)

## Descrição
Dado um array de inteiros `nums` e um inteiro `k`, determine o **número mínimo de operações** necessárias para que a soma dos elementos do array seja divisível por `k`.

Em uma operação, é possível diminuir qualquer elemento do array em `1`.

## Solução (Python)

```python
class Solution:
    def minOperations(self, nums: List[int], k: int) -> int:
        return sum(nums) % k
