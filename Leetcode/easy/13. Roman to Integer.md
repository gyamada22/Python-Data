# 13. Roman to Integer

## Descrição
Escreva uma solução para converter um número romano em um inteiro.

A regra principal dos números romanos é:
- Normalmente os valores são **somados** da esquerda para a direita.
- Porém, quando um símbolo de **menor valor vem antes de um de maior valor**, ele deve ser **subtraído**.

Exemplos:
- `III` → 3  
- `IV` → 4 (5 - 1)  
- `IX` → 9 (10 - 1)  
- `LVIII` → 58  
- `MCMXCIV` → 1994  

---

## Solução (Python 3)

```python
class Solution:
    def romanToInt(self, s: str) -> int:
        values = {
            'I': 1, 'V': 5, 'X': 10, 'L': 50,
            'C': 100, 'D': 500, 'M': 1000
        }
        
        total = 0
        n = len(s)
        
        for i in range(n):
            # Se o valor atual for menor que o próximo, subtrai
            if i < n - 1 and values[s[i]] < values[s[i + 1]]:
                total -= values[s[i]]
            else:
                total += values[s[i]]
                
        return total
