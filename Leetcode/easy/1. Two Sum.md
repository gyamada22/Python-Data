# 1. Two Sum

**Data:** 08/01/2026  
**Dificuldade:** Easy  
**Tópicos:** Python3, Array, Hash Table, Enumeration  
**Link:** [Two Sum - LeetCode](https://leetcode.com/problems/two-sum/description/)

## Descrição
Encontre **dois índices em uma lista `nums`** cujos valores somem exatamente ao `target`.  
Retorne os índices como uma lista `[index1, index2]`.  

## Solução (Python3)

```python
class Solution:
    def twoSum(self, nums: list[int], target: int) -> list[int]:
        lookup = {} 
        for i, n in enumerate(nums):
            complement = target - n
            if complement in lookup:
                return [lookup[complement], i]
            lookup[n] = i
